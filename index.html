# קבוצה מש׳ נצ׳ ת׳ הפ׳ שערים הפרש נק׳
איך זה עובד
  1. האתר טוען נתונים מ-data.json.
  2. GitHub Actions מעדכן את data.json פעם ביום רק אם יש שינוי.
  3. אין עריכה דרך האתר — הכל קריאה בלבד.
כדי שה-Action יוכל לבצע Commit אוטומטי, צריך לאפשר: Settings → Actions → General → Workflow permissions → Read and write.
